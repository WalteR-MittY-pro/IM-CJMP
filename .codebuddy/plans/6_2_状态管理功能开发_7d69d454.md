## 产品概述

在telegram_cjmp工程中实现状态管理功能，用于显示消息发送状态和联系人在线状态

## 核心功能

1. **消息状态显示**：发送中(时钟图标) → 已发送(单勾) → 已送达(双勾灰色) → 已读(双勾蓝色) | 失败(红色感叹号)
2. **在线状态显示**：在线(绿色) / 离线 / 最近在线(X分钟前) / 正在输入(typing)

## 修改目标

- MockData.cj: Message结构添加status字段，ContactItem添加onlineStatus字段
- ChatDetailPage.cj: 发送消息后显示状态图标变化
- ContactsPage.cj: 显示联系人在线状态（绿点/灰点/时间/正在输入）

## 技术栈

- 框架：Cangjie UI (仓颉UI)
- 语言：Cangjie

## 技术架构

- 数据驱动UI：状态字段控制UI显示
- 消息状态枚举：sending → sent → delivered → read | failed
- 在线状态枚举：online / offline / recently / typing

## 实现方案

1. **Message结构扩展**：添加status字段(String类型)，支持sending/sent/delivered/read/failed状态
2. **ContactItem结构扩展**：添加onlineStatus字段(String类型)，支持online/offline/recently/typing状态
3. **ChatDetailPage消息状态显示**：发送消息时显示时钟 → 1秒后变单勾 → 2秒后变双勾灰色 → 对方读取后变双勾蓝色
4. **ContactsPage在线状态显示**：根据onlineStatus显示对应状态文本和颜色

## 实现细节

- 消息状态变化：发送新消息时初始为sending，使用setTimeout模拟状态演变
- 在线状态显示：online显示绿色圆点+在线文字，offline显示灰色圆点+最后在线时间，recently显示灰色圆点+具体时间，typing显示"正在输入..."
- 颜色定义：在线绿色(0x4CAF50)，离线灰色(0x999999)，失败红色(0xF44336)
