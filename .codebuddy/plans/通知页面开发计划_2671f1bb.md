## Product Overview

在Telegram设置页面中新增通知(Notifications)详情页，管理消息通知和声音设置。

## Core Features

- 3.1 顶部导航栏：返回按钮 + 标题"通知"
- 3.2 功能列表：
- 通知分组：消息通知(Toggle)、群组通知(Toggle)、私密通知(Toggle)
- 声音分组：声音(默认>)、通知音(默认>)
- 振动分组：振动(默认>)、重复提示(1次>)
- 显示分组：显示预览(Toggle)、通知横幅(Toggle)
- 例外：自定义通知(数量>)
- 3.3 布局结构：按照设计规范实现完整的列表布局

## Component Specifications

- 列表项高度：56dp
- 分割线：1dp #EEEEEE
- 分组标题：14dp #999999
- Toggle开关：主色 #2196F3

## Tech Stack

- Framework: 仓颉CJMP + ArkUI
- Language: Cangjie
- 构建工具: keels

## Implementation Approach

1. 创建 NotificationsPage.cj 通知详情页面

- 使用 Stack 布局实现页面覆盖
- 使用 Column + Scroll + List 实现功能列表
- 使用 Checkbox 组件实现 Toggle 开关（参考PrivacySecurityPage）
- 使用 @State 装饰器管理通知设置状态

2. 修改 SettingsPage.cj 添加跳转逻辑

- 为"通知"列表项添加 onClick 事件
- 在 Stack 中添加 NotificationsPage 条件渲染

## Implementation Notes

- 复用 PrivacySecurityPage.cj 的代码结构和模式
- 使用 Unicode 表情符号作为图标（参考现有代码）
- 保持与现有页面一致的代码风格和组件规范
