## 1. 主界面（聊天列表）

### 1.1 界面组成

#### 1.1.1 导航栏区域

**左侧按钮**
- 功能：打开侧边菜单/抽屉
- 触发：点击
- 行为：从左侧滑出菜单，显示设置、帮助等选项

**标题区域**
- 显示：应用名称 "Telegram" 或当前文件夹名称
- 触发：点击（部分版本）
- 行为：切换文件夹或显示文件夹列表

**右侧按钮组**
- 搜索按钮：
  - 触发：点击
  - 行为：展开搜索栏，聚焦输入框
- 菜单按钮（⋮）：
  - 触发：点击
  - 行为：弹出下拉菜单
  - 选项：新建群组、新建频道、联系人、通话、已保存消息、设置

#### 1.1.2 搜索栏

**显示状态**
- 隐藏：默认状态
- 展开：点击搜索按钮后显示

**功能**
- 实时搜索：输入时即时过滤结果
- 搜索范围：
  - 聊天名称
  - 最后消息内容
  - 全局消息（点击"全局搜索"）
- 清除按钮：快速清空输入
- 取消按钮：关闭搜索，恢复列表

**交互**
- 聚焦时：软键盘弹出，搜索结果实时更新
- 失焦时：点击取消或返回键关闭

#### 1.1.3 聊天列表

**列表项组成**
```
[头像] [名称             时间  ]
       [最后消息预览    [角标]]
```

**显示信息**
- 头像：
  - 个人聊天：用户头像（圆形）
  - 群组：群组头像（圆形）
  - 频道：频道头像（圆形）
  - 在线状态：绿色边框（个人聊天）
- 名称：
  - 个人：用户名称
  - 群组：群组名称
  - 频道：频道名称 + 验证标识（✓）
  - 颜色：黑色（默认）/ 蓝色（未读）
- 时间：
  - 格式：刚刚 / 5分钟前 / 昨天 / 日期
  - 位置：右上角
- 最后消息：
  - 发送者（群组中）：发送者名称 + ":"
  - 内容预览：文本/附件类型/草稿
  - 草稿提示：红色 "草稿:" 前缀
  - 颜色：灰色（默认）/ 蓝色（提及）
- 状态图标：
  - 已读：蓝色双勾
  - 已送达：灰色双勾
  - 已发送：单勾
  - 发送中：时钟图标
  - 发送失败：红色感叹号
- 未读角标：
  - 位置：右下角
  - 显示：未读数量或圆点
  - 颜色：蓝色（默认）/ 灰色（静音）
- 固定图标：
  - 位置：右侧（时间下方）
  - 显示：图钉图标
- 静音图标：
  - 位置：名称后
  - 显示：静音图标
- 正在输入：
  - 显示："正在输入..."（替代最后消息）
  - 动画：三个点跳动

**列表操作**

1. **点击聊天项**
   - 触发：单击整个列表项
   - 行为：打开对应的聊天界面
   - 动画：从右滑入新页面

2. **向右滑动**
   - 触发：从左向右滑动列表项
   - 距离阈值：80pt
   - 行为选项：
     - 固定/取消固定
     - 归档
     - 标记已读/未读
   - 显示：滑动时显示操作按钮
   - 快速滑动：直接触发默认操作（固定）

3. **向左滑动**
   - 触发：从右向左滑动列表项
   - 距离阈值：80pt
   - 行为选项：
     - 静音/取消静音
     - 删除
     - 更多选项
   - 显示：多个操作按钮（根据滑动距离逐个显示）
   - 快速滑动：直接触发删除（需确认）

4. **长按聊天项**
   - 触发：长按列表项 0.5秒
   - 行为：弹出上下文菜单
   - 菜单选项：
     - 固定/取消固定
     - 归档
     - 静音（1小时/8小时/2天/永久）
     - 标记已读/未读
     - 清除历史
     - 删除聊天
     - 查看信息
   - 触觉反馈：震动

5. **多选模式**
   - 进入方式：
     - 长按后移动到其他项
     - 点击菜单的"选择消息"
   - 行为：
     - 左侧显示复选框
     - 选中项高亮显示
     - 底部显示操作栏
   - 操作栏按钮：
     - 归档（支持批量）
     - 静音（支持批量）
     - 标记已读（支持批量）
     - 删除（支持批量）
   - 退出：点击取消或返回键

#### 1.1.4 列表行为

**下拉刷新**
- 触发：列表顶部下拉
- 距离：60pt 触发
- 动画：
  - 下拉阶段：箭头图标拉伸
  - 刷新阶段：转圈动画
  - 完成阶段：收起动画
- 行为：同步最新聊天列表

**无限滚动**
- 触发：滚动到列表底部
- 行为：自动加载更多历史聊天
- 显示：底部加载指示器

**快速滚动**
- 触发：快速拖动滚动条
- 显示：右侧日期提示气泡
- 行为：快速定位到指定日期

**智能滚动**
- 新消息到达：
  - 当前在顶部：自动滚动到新消息位置
  - 当前在底部：保持位置不变
- 未读消息：
  - 显示"向下滚动"浮动按钮
  - 点击跳转到第一条未读

#### 1.1.5 文件夹/筛选标签

**文件夹标签栏**
- 位置：列表顶部（搜索栏下方）
- 显示：横向滑动的标签列表
- 默认标签：
  - 全部聊天
  - 未读
  - 个人
  - 群组
  - 频道
  - 机器人
  - 自定义文件夹（用户创建）

**标签交互**
- 点击标签：切换到对应的筛选视图
- 长按标签：编辑或删除文件夹
- 滑动标签栏：查看更多文件夹

**文件夹规则**
- 包含类型：个人/群组/频道/机器人
- 包含聊天：指定的聊天列表
- 排除聊天：排除的聊天列表
- 未读优先：未读消息置顶

#### 1.1.6 底部标签栏（Tab Bar）

**标签项**
1. 联系人（Contacts）
   - 图标：人形图标
   - 点击：跳转到联系人列表
   
2. 通话（Calls）
   - 图标：电话图标
   - 点击：跳转到通话记录
   - 角标：未接来电数量
   
3. 聊天（Chats）
   - 图标：消息图标
   - 状态：默认选中
   - 角标：总未读消息数
   
4. 设置（Settings）
   - 图标：齿轮图标
   - 点击：跳转到设置页面

**标签状态**
- 选中：图标填充 + 蓝色 + 标题显示
- 未选中：图标轮廓 + 灰色
- 角标：红色圆点或数字

### 1.2 特殊功能

#### 1.2.1 归档文件夹

**显示**
- 位置：列表顶部（有归档时）
- 样式：特殊的归档入口项
- 显示内容：
  - 归档图标
  - "归档" 标题
  - 归档中的聊天数量
  - 未读数量（如有）

**交互**
- 点击：展开归档聊天列表
- 向下滑动：隐藏归档入口
- 向上滑动：显示归档入口

**自动取消归档**
- 触发：归档的聊天收到新消息
- 行为：聊天自动从归档移回主列表
- 可配置：设置中可关闭此功能

#### 1.2.2 固定聊天

**固定位置**
- 位置：列表最顶部
- 数量限制：
  - 普通用户：5个
  - Premium 用户：10个

**显示**
- 固定图标：列表项右侧
- 分隔线：固定聊天和普通聊天之间

**排序**
- 拖动排序：长按固定聊天可拖动调整顺序
- 固定时间：按固定时间倒序

#### 1.2.3 聊天文件夹

**创建文件夹**
- 入口：设置 → 文件夹
- 步骤：
  1. 命名文件夹
  2. 选择图标
  3. 设置包含规则
  4. 设置排除规则
- 保存：自动同步到云端

**编辑文件夹**
- 入口：长按文件夹标签
- 操作：
  - 重命名
  - 更改图标
  - 调整规则
  - 删除文件夹

**文件夹共享**
- 功能：生成文件夹邀请链接
- 行为：他人点击链接自动创建相同规则的文件夹

## 2. 聊天界面

### 2.1 界面组成

#### 2.1.1 导航栏区域

**左侧返回按钮**
- 图标：左箭头 + 聊天名称（缩略）
- 点击：返回聊天列表
- 滑动手势：从左边缘向右滑动返回

**中间信息区域**
- 点击：跳转到聊天详情页
- 显示内容：
  - 个人聊天：
    - 名称
    - 在线状态（在线 / 最后在线时间）
  - 群组聊天：
    - 群组名称
    - 成员数量（X 位成员）
  - 频道：
    - 频道名称
    - 订阅者数量（X 位订阅者）
- 实时更新：
  - 正在输入：显示 "正在输入..."
  - 录音中：显示 "正在录制语音消息..."
  - 上传中：显示 "正在发送照片..."

**右侧按钮组**
1. 通话按钮（个人聊天）
   - 图标：电话图标
   - 点击：弹出选择菜单（语音通话 / 视频通话）
   
2. 搜索按钮
   - 图标：放大镜
   - 点击：打开聊天内搜索

3. 更多按钮（⋮）
   - 点击：弹出菜单
   - 菜单选项：
     - 查看信息
     - 静音/取消静音
     - 选择消息
     - 清除历史
     - 删除聊天
     - 报告（陌生人）

#### 2.1.2 消息列表区域

**列表行为**
- 显示方向：从底部向上（时间倒序）
- 初始位置：最底部（最新消息）
- 滚动方向：双向无限滚动
  - 向上滚动：加载历史消息
  - 向下滚动：回到最新消息

**消息分组**
- 日期分隔：
  - 显示：日期标签（居中，圆角背景）
  - 内容：今天 / 昨天 / 具体日期
- 消息合并：
  - 条件：同一人，时间间隔 < 5分钟
  - 效果：
    - 省略头像（除第一条）
    - 圆角调整（中间消息顶部圆角变小）
    - 减少间距

**滚动控制**
1. 滚动到底部按钮
   - 显示条件：当前位置距底部 > 1屏
   - 位置：右下角浮动
   - 点击：平滑滚动到最新消息
   - 角标：未读消息数（如有）

2. 提及按钮
   - 显示条件：有未读的@提及消息
   - 位置：滚动到底部按钮上方
   - 点击：跳转到下一条提及消息
   - 角标：未读提及数量

3. 快速跳转
   - 触发：点击回复的引用
   - 行为：滚动并高亮原消息
   - 动画：平滑滚动 + 闪烁高亮

**加载状态**
- 向上加载历史：
  - 显示：顶部加载指示器
  - 行为：保持当前消息位置不变
- 同步最新消息：
  - 显示：顶部下拉刷新
  - 行为：刷新后自动滚动到底部

#### 2.1.3 消息气泡

**消息类型与显示**

1. **文本消息**
   - 内容：纯文本或富文本
   - 链接：自动识别并可点击
   - 格式化：支持 Markdown（粗体、斜体、代码）
   - 表情：自动放大（仅表情时）
   - 编辑标识：消息右下角显示 "已编辑"

2. **图片消息**
   - 显示：缩略图（保持比例）
   - 加载：渐进式加载（模糊→清晰）
   - 占位：尺寸占位符
   - 说明文字：图片下方可选文字
   - 批量：多图组合显示（网格布局）

3. **视频消息**
   - 显示：视频封面 + 播放按钮
   - 时长：右下角显示时长
   - 进度：下载/上传进度环
   - 自动播放：静音自动播放（可配置）

4. **语音消息**
   - 显示：播放按钮 + 波形图 + 时长
   - 播放状态：
     - 未播放：蓝色波形
     - 播放中：播放进度 + 当前时间
     - 已播放：灰色波形
   - 速度：点击速度按钮（1x/1.5x/2x）
   - 连续播放：播放完自动播放下一条

5. **文件消息**
   - 显示：文件图标 + 文件名 + 大小
   - 状态：
     - 未下载：下载按钮
     - 下载中：进度环 + 暂停按钮
     - 已下载：打开按钮
   - 类型图标：根据文件类型显示不同图标

6. **位置消息**
   - 显示：地图缩略图
   - 信息：地点名称 + 地址
   - 点击：打开完整地图

7. **联系人消息**
   - 显示：头像 + 姓名 + 手机号
   - 按钮：添加联系人 / 发送消息

8. **投票消息**
   - 显示：问题 + 选项列表
   - 状态：
     - 未投票：可点击选项
     - 已投票：显示百分比和票数
   - 类型：单选 / 多选 / 测验

**消息状态**

显示位置：消息右下角（发送消息）

```
状态图标：
- 发送中：⏱ 时钟图标（灰色）
- 已发送：✓ 单勾（灰色）
- 已送达：✓✓ 双勾（灰色）
- 已读：✓✓ 双勾（蓝色/绿色）
- 失败：❗ 感叹号（红色）
```

点击失败图标：
- 弹出菜单：重新发送 / 删除

**消息时间**

显示位置：
- 独立消息：消息气泡右下角内嵌
- 合并消息：鼠标悬停显示

格式：
- 今天：HH:MM
- 昨天：昨天 HH:MM
- 更早：MM-DD HH:MM

#### 2.1.4 消息交互

**1. 点击消息**

根据消息类型不同：
- 文本消息：选中文本（长按）
- 图片消息：打开图片查看器
- 视频消息：播放视频
- 语音消息：播放/暂停
- 文件消息：打开文件
- 链接：打开链接预览或浏览器

**2. 长按消息**

触发：长按消息 0.5秒
行为：弹出上下文菜单
菜单选项：
- 回复
- 编辑（自己的消息，48小时内）
- 复制（文本消息）
- 转发
- 固定消息
- 删除（自己 / 双方）
- 选择
- 报告（陌生人消息）

**3. 向右滑动消息**

触发：从左向右滑动消息气泡
阈值：30pt
行为：触发回复操作
动画：回复图标和滑动距离联动
松手：
- 超过阈值：进入回复模式
- 未超过阈值：弹回原位

**4. 双击消息**

触发：快速双击消息（部分版本）
行为：快速反应（点赞 ❤️）
动画：心形图标从消息中心放大并飞出

**5. 点击消息内元素**

- 点击链接：打开链接（内置浏览器或外部）
- 点击提及（@用户）：查看用户信息
- 点击话题（#标签）：搜索相关消息
- 点击回复引用：跳转到原消息
- 点击电话号码：拨打电话
- 点击邮箱：发送邮件

**6. 选择模式**

进入方式：
- 点击菜单的"选择消息"
- 长按消息后选择"选择"

行为：
- 左侧显示复选框
- 点击消息切换选中状态
- 顶部显示已选数量
- 底部显示操作栏

操作栏按钮：
- 转发：转发选中的消息
- 复制：复制文本内容
- 删除：删除选中的消息
- 更多：更多操作（分享、报告等）

退出：点击取消或返回键

#### 2.1.5 输入区域

**输入框组成**
```
[附件] [输入框...            ] [语音/发送]
```

**输入框功能**

1. **文本输入**
   - 占位符："Message"
   - 自动增长：1行 → 4行（最大）
   - 自动聚焦：从通知打开时
   - 键盘类型：文本键盘
   - 表情按钮：左侧表情图标

2. **草稿保存**
   - 自动保存：输入时自动保存草稿
   - 云端同步：草稿同步到其他设备
   - 恢复：下次打开自动恢复草稿
   - 清除：发送后自动清除草稿

3. **格式化工具栏**（可选显示）
   - 触发：选中文本后显示
   - 选项：
     - 粗体（**text**）
     - 斜体（*text*）
